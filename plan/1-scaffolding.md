# Scaffolding.

Steps:

1. Set up the index.html page (I have created a basic template). CSS resets live in the reset.css file and styles for the game live in styles.css. JS logic lives in app.js.

2. In the main.css file: define font family as `font-family: Menlo, Consolas, Monaco, Adwaita Mono, Liberation Mono, Lucida Console, monospace;`. Use this for all text on the page.

3. Create a basic 4x4 grid using css grid. Use a medium lightness beige for the gap (you can use background-color for this). For the grid cells use a lighter beige. Define these as color variables for consistency/reusability.

4. Set up a current score and best score headers at the top. We will populate these later.

Requirements:
- Size of the grid is 4x4
- Use css grid
- Above the grid, display the score
